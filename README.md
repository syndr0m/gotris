Description
===========

Tetris-like in golang (POC)

Install
===========

Prerequisites
---------------

you need go, git and mercurial installed

Installing dependencies
---------------

```
go get -u github.com/google/gxui/...
go get -u github.com/syndr0m/mygameengine/...
```

Build
-------

```
go build
```
